# ⏱️ Stopwatch Web Application

## 🚀 Task-02 | Prodigy InfoTech Web Development Internship

This project is a fully functional Stopwatch Web Application developed using HTML, CSS, and JavaScript.

The application allows users to accurately track time intervals, record lap times, and control the stopwatch through an interactive user interface.

---

## 📌 Features

✅ Start Stopwatch

✅ Pause Stopwatch

✅ Reset Stopwatch

✅ Record Multiple Lap Times

✅ Real-Time Time Tracking

✅ Responsive User Interface

✅ Interactive and User-Friendly Design

## ⚡ How It Works

1. Click **Start** to begin timing.
2. Click **Pause** to stop the timer temporarily.
3. Click **Lap** to record the current elapsed time.
4. Click **Reset** to clear the timer and lap records.
